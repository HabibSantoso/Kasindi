# Final Project Pengembangan Aplikasi Mobile 2024

## Project Name
### Kasindi

## Project Description
### Aplikasi ini di buat untuk memeudahkan Finance sebuah perusahaan dalam menenjemen dan monitoring pembukuan keuangan perusahaan menggunakan mobile. Sehingga meminimalisir terjadinya kesalahan dalam pembukuan perusahaan.

## Flow
### User: Admin, Bendahara
### Fitur: Create akun(admin), Login(admin, Bendahara), view Daftar transaksi(admin, Bendahara), view Daftar user(admin), Edit user(admin), Delete User(admin), Input laporan transaksi(Bendahara), Detail Laporan transaksi(Bendahara), Edit Laporan transaksi(Bendahara), Delete Laporan transaksi(Bendahara),  Logout(admin, Bendahara)

### 1. SplashScreen -> Login ->  Home -> Create/Update/Delete Laporan transaksi -> Logout (Bendahara)
### 2. SplashScreen -> Login ->  Home -> Daftar user -> Create/Update/Delete User -> Logout (Admin)
